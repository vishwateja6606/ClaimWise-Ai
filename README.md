# ClaimWise AI

ClaimWise AI is an AI-powered insurance claim processing system built with Java, Spring Boot, PostgreSQL, Kafka, Redis, Docker, and Spring AI.

## Goal

Customers can submit insurance claims, upload documents, and track claim status. Claim officers can review claims with AI-generated summaries, missing document detection, fraud-risk suggestions, and recommended actions.

## Planned Tech Stack

- Java 21
- Spring Boot
- Spring Security + JWT
- PostgreSQL
- Kafka
- Redis
- Spring AI
- Docker
- Swagger/OpenAPI
- JUnit and Mockito
